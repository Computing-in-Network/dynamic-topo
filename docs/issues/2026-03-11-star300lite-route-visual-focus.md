# star300lite 静态路由前端聚焦展示增强

## 背景

当前前端已经能读取控制器最近一次成功下发的静态路由快照，并展示：

- 快照状态
- 快照时间
- 已下发帧号
- 第一跳
- 完整路径
- 球面链路高亮

但实际演示时，原始拓扑链路数量仍然较多，单纯高亮当前路径不够醒目：

- 观众很难在全局链路背景中快速识别目标路径
- 截图和汇报画面不够聚焦
- 单纯换颜色提升有限

## 目标

增强静态路由前端展示效果，让当前选中的路由在页面上更容易被识别，适合现场演示和截图汇报。

## 交付范围

1. 路由视觉强化
   - 提升路径链路颜色亮度
   - 增加路径链路宽度
   - 加强起点、终点、路径节点的颜色区分

2. 路由展示模式
   - 增加三种展示模式：
     - 普通
     - 显示路由
     - 只看路由

3. 聚焦规则
   - `显示路由` 模式下：
     - 非路径链路明显变暗
     - 非路径节点明显变暗
   - `只看路由` 模式下：
     - 非路径链路隐藏
     - 非路径节点隐藏
     - 页面只保留当前路径相关内容

4. 面板联动
   - 在静态路由演示面板中展示当前模式
   - 模式切换按钮应适合直接用于演示操作

## 验收标准

1. 路由路径比普通链路明显更亮、更粗
2. `显示路由` 模式下，当前路径能一眼识别
3. `只看路由` 模式下，页面只保留当前路径相关节点和链路
4. 前端构建通过，且不影响原有快照读取逻辑

## 说明

这次改动不涉及路由控制逻辑，不改变已下发路径本身，只增强前端展示层的可读性和演示效果。
